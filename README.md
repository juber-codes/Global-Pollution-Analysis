# 🌍 Global Pollution Analysis using Machine Learning

This project focuses on analyzing global pollution data using Machine Learning techniques such as Linear Regression and Logistic Regression. The goal is to understand pollution trends, predict pollution levels, and classify pollution severity.

---

## 📌 Features

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Visualization of pollution trends  
- Linear Regression for pollution level prediction  
- Logistic Regression for pollution classification (e.g., High/Low)  
- Model evaluation and performance metrics  
- Feature importance analysis  

---

## 🛠️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📊 Dataset

- Global pollution dataset (Air Quality, CO2 emissions, etc.)  
- Contains features like:
  - Country  
  - Year  
  - CO2 Emissions  
  - Air Quality Index (AQI)  
  - Industrial Emissions  
  - Population  

---

## 🚀 Project Workflow

1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Model Building  
   - Linear Regression  
   - Logistic Regression  
6. Model Evaluation  
7. Prediction & Results  

---

## 📈 Machine Learning Models

### 🔹 Linear Regression
- Used to predict continuous pollution values (e.g., CO2 levels)

### 🔹 Logistic Regression
- Used to classify pollution levels (High / Low)

---

## 📊 Evaluation Metrics

- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- R² Score  
- Accuracy (for Logistic Regression)  
- Confusion Matrix
